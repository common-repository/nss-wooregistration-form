== Descriptions ==

If you have woo-commerce/eCommerce business & integrated with WordPress. This is a simple woo-commerce user registration plugin you can use it easily. 

Go to the woo-commerce settings->account/privacy tab then select a checkbox which is "Allow customers to create an account on the "My Account page". It will be automatically added a my-account page which is so easy nothing to create a new page. So you can just active and open my-account page.

<h2>How to use Recaptcha?</h2>

Go to this <a href="https://www.google.com/recaptcha/admin">Google Recaptcha</a>. Firstly register your site domain then create a new 'site key' and 'secret key'. See our screenshot.

I think it will help your projects for new user registration. :)

== Installation ==

*. upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the wordpress plugins screen directly.
*. Activate the plugin through the 'Plugins' screen in wordPress
*. Use the Settings->plugin Name screen to configure the plugin
